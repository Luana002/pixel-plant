# 🌱 Pixel Plant
> Nome provisório do projeto. Alterações poderão ser realizadas durante o desenvolvimento.
> Uma plataforma web com temática Pixel Art para organizar e gerenciar coleções de plantas em um jardim digital personalizado.


---

## 📖 Sobre o Projeto

O **Pixel Plant** transforma o cuidado e a organização de plantas em uma experiência digital inspirada no Pixel Art. A plataforma permite que usuários construam seu próprio jardim virtual, registrando e gerenciando suas plantas em um ambiente agradável, organizado e personalizado.

---

## 🎯 Objetivo

Desenvolver uma plataforma que permita aos usuários criar uma conta, acessar um ambiente personalizado e gerenciar informações sobre suas plantas de forma prática e centralizada.

---

## 👥 Público-Alvo

O Pixel Plant é destinado a pessoas que cultivam ou colecionam plantas e buscam uma forma prática de registrar, organizar e acompanhar sua coleção em um ambiente digital personalizado.

## ✨ Funcionalidades

### MVP (Versão Inicial)

#### Autenticação de Usuários

* Cadastro de usuário;
* Login;
* Logout.

#### Gerenciamento de Plantas

* Cadastro de plantas;
* Visualização de plantas cadastradas;
* Edição de informações das plantas;
* Exclusão de plantas.

#### Dashboard

* Área pessoal do usuário;
* Visualização da coleção de plantas cadastradas.

## 📋 Requisitos Funcionais

* O usuário deve ser capaz de criar uma conta.
* O usuário deve ser capaz de realizar login e logout.
* O usuário deve ser capaz de cadastrar plantas.
* O usuário deve ser capaz de visualizar suas plantas cadastradas.
* O usuário deve ser capaz de editar informações de uma planta.
* O usuário deve ser capaz de excluir uma planta.

---

## 🔒 Requisitos Não Funcionais

* Interface responsiva para diferentes dispositivos.
* Navegação simples e intuitiva.
* Armazenamento seguro dos dados dos usuários.
* Boa performance nas operações de cadastro e consulta.
* Design inspirado em Pixel Art.
